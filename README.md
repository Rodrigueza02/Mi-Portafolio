
# 🎨 Portafolio Web - Maria Rodríguez

## 📌 Descripción

Este proyecto es un portafolio web desarrollado con **Next.js y Tailwind CSS**, diseñado para presentar mis proyectos de desarrollo frontend y full stack.

El portafolio está inspirado en una estética moderna con temática dinámica, incluyendo animaciones, efectos visuales y una estructura organizada por secciones para facilitar la navegación.

---

## 🚀 Características

- Diseño moderno y responsivo 📱💻
- Navegación entre páginas (Inicio, Sobre mí, Proyectos)
- Sistema de búsqueda dentro del portafolio 🔍
- Vista previa interactiva de proyectos 👁️
- Efectos visuales y animaciones personalizadas
- Integración con GitHub para visualizar proyectos

---

## 🛠️ Tecnologías utilizadas

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Lucide Icons**

## 📌 Proyectos Destacados

### 🎵 Sagiria - Reproductor Musical
Aplicación web conectada a la API de Spotify que permite:
- Reproducir música en tiempo real
- Gestionar playlists
- Marcar favoritos
- Buscar canciones

**Tecnologías:** Flask, JavaScript, HTML, CSS, Spotify API

---

### 🎨 Mentes Creativas - Plataforma Educativa Infantil
Aplicación interactiva para niños de primaria que incluye:
- Sistema solar 🌍
- Pintura 🎨
- Escultura ✏️

**Tecnologías:** React, Vite, TypeScript, Tailwind, Jest

## ⚙️ Instalación y uso

Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-PORTAFOLIO.git
cd TU-PORTAFOLIO
````

Instalar dependencias:

```bash
npm install
```

Ejecutar en desarrollo:

```bash
npm run dev
```

Abrir en el navegador:

```
http://localhost:3000
```
